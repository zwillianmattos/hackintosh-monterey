# Hackintosh Monterey

## Hardware
![Setup Info](./setup.png)

```
Intel i5 9400f
RX580 8gb
48GB RAM
SSD M2 512gb
```

## Install

1. Format USB to FAT32;
2. Run install.sh;
3. Copy folders `EFI` and `com.apple.recovery.boot` to USB;
4. Enjoy;
